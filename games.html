The Hungry Bunny

A simple and fun Single Screen 2D game made using Unity and C#. The player is a hungry bunny that goes around to eat all the carrots on the farm. Move around using the arrow keys and avoid the bouncing red monsters! Art done using a mouse on paint so don't expect too much...

Currently redesigning the graphics of the game. Also hope to create versions that will work on a Mac OS and cell phones. Stay tuned for newer versions!

Download and Play (Windows Only)
